# TicketCore (Online Ticket Booking Platform)

## Purpose
TicketCore is a centralized online ticket booking platform where users can easily discover and book travel tickets for various transport modes including Bus, Train, Launch, and Flight. It bridges the gap between vendors adding transport services and users booking them seamlessly.

## Live URL
[https://online-ticket-booking-platform-gamma.vercel.app/](https://online-ticket-booking-platform-gamma.vercel.app/)

## Key Features
- **Role-based Dashboards:** Dedicated dashboards for Admin, Vendor, and User.
- **Secure Authentication:** Email/password and Google social login powered by BetterAuth.
- **Ticket Management:** Vendors can add, update, and manage transport tickets.
- **Booking & Checkout:** Users can book tickets, track countdowns, and pay securely via Stripe.
- **Fraud Prevention:** Admins can monitor, advertise tickets, and permanently ban fraudulent vendors.
- **Modern UI:** Clean, responsive, full-dark mode compatible aesthetics.
